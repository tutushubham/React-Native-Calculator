# React-Native-Calculator
================

Introduction
------------

React Native is an open-source mobile application framework created by Facebook, Inc. It is used to develop applications for Android, Android TV, iOS, macOS, tvOS, Web, Windows and UWP by enabling developers to use React's framework along with native platform capabilities.

This sample shows how to use the libraries to build
a complete basic calculator app that implements the reaccct native.

Pre-requisites
--------------

* NodeJS, React Native and VSCode installed

* Make sure Android Studio is updated, as well as your SDK and Gradle.
Otherwise, you may have to wait for a while until all the updates are done.

* A device or emulator that runs SDK level 20

You need to be solidly familiar with the JavaScript programming language,
object-oriented design concepts, and Android Native Development Fundamentals.

Getting Started
---------------

1. npm start
2. Scan the QR code using Expo App
2. Run and try the app
3. Enjoy and Contribute!
